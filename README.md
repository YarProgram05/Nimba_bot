# Nimba_bot

Telegram-бот для работы с данными маркетплейсов Ozon и Wildberries.

Проект помогает собирать и формировать отчеты по продажам и остаткам, объединять данные по нескольким кабинетам, а также автоматизировать регулярную отправку отчетов в чат.

Основные возможности:
- отчеты по продажам Ozon и Wildberries;
- отчеты по остаткам товаров;
- сводный отчет по нескольким маркетплейсам;
- автоматические отчеты по расписанию;
- генерация штрихкодов и PDF-этикеток из Excel;
- конвертация CSV в XLSX.

Код проекта находится в папке `market_bot/`.
